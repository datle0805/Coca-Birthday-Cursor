# My Next.js App

This is a Next.js application configured with Zustand for state management, Axios for API calls, Material-UI (MUI) for UI components, and Tailwind CSS for styling. The project also includes Jest for testing, along with Prettier and ESLint for code formatting and linting.

## Getting Started

To get started with this project, follow the instructions below.

### Prerequisites

Make sure you have the following installed:

- Node.js (version 14 or later)
- npm or yarn

### Installation

1. Clone the repository:

   ```
   git clone <repository-url>
   cd my-nextjs-app
   ```

2. Install the dependencies:

   ```
   npm install
   ```

   or

   ```
   yarn install
   ```

### Running the Development Server

To start the development server, run:

```
npm run dev
```

or

```
yarn dev
```

Your application will be available at `http://localhost:3000`.

### Building for Production

To build the application for production, run:

```
npm run build
```

or

```
yarn build
```

### Running Tests

To run the tests, use:

```
npm test
```

or

```
yarn test
```

### Code Formatting

To format the code using Prettier, run:

```
npm run format
```

or

```
yarn format
```

### Linting

To lint the code using ESLint, run:

```
npm run lint
```

or

```
yarn lint
```

## Folder Structure

- `src/app`: Contains the main application layout and pages.
- `src/components`: Contains reusable components.
- `src/pages/api`: Contains API routes.
- `src/store`: Contains Zustand store setup.
- `src/hooks`: Contains custom hooks.
- `src/types`: Contains TypeScript types and interfaces.
- `src/utils`: Contains utility functions for API calls.
- `tests`: Contains Jest tests.
